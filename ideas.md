## Idea A: Dark Editorial Minimalism

**Design Movement:** Post-Digital Editorial / Swiss Modernism meets Dark Web Aesthetics

**Core Principles:**
- Near-black background with surgical whitespace — every element earns its place
- Typography as the primary visual hierarchy tool, not imagery
- Asymmetric tension: large type anchors left, negative space breathes right
- Monochromatic palette punctuated by a single teal accent

**Color Philosophy:**
- Base: #0D0D0D (near-black) — conveys authority and focus
- Surface: #141414 (card backgrounds) — subtle depth without harsh contrast
- Text: #F0EDE8 (warm white) — softer than pure white, more editorial
- Accent: #1A6B5F (deep teal) — professional, distinctive, not overdone
- Muted: #4A4A4A — for secondary text and dividers

**Layout Paradigm:**
- Full-bleed dark sections with generous vertical rhythm (120px+ section padding)
- Hero: left-aligned text block with name in oversized display type, status badge floating above
- Portfolio: 3-column masonry grid with hover overlays
- Asymmetric two-column layouts for About/Services

**Signature Elements:**
- Large section numbers (01, 02, 03...) in faint teal, positioned top-left of each section
- Thin horizontal rule dividers with section labels
- Pill-shaped status badge with pulsing green dot

**Interaction Philosophy:**
- Hover states reveal information (portfolio card title overlays)
- CTA button has a subtle fill-from-left animation on hover
- Smooth scroll with section snap points

**Animation:**
- Fade-up entrance animations on scroll (staggered for card grids)
- Portfolio cards: image scale 1.05 + dark overlay + title fade-in on hover
- CTA button: background slides in from left on hover

**Typography System:**
- Display: Cormorant Garamond (700) — editorial, distinctive, high-contrast
- Heading: DM Serif Display — slightly warmer, pairs well with Cormorant
- Body: DM Sans (400, 500) — clean, modern, highly legible
- Monospace accents: JetBrains Mono for section numbers/labels

## Idea B: Brutalist Confidence

**Design Movement:** Digital Brutalism / Neo-Grotesque

**Core Principles:**
- Raw, unapologetic typography — oversized, cropped, intentionally overwhelming
- Grid breaks as a design feature, not an accident
- High-contrast black/white with a single electric accent
- Structure exposed: borders, rules, and grids are visible design elements

**Color Philosophy:**
- Base: #F5F2EC (warm off-white) — paper-like, tactile
- Text: #0D0D0D — maximum contrast
- Accent: #D97706 (amber) — warm, energetic, memorable
- Borders: #0D0D0D — thick, visible, structural

**Layout Paradigm:**
- Oversized hero type that breaks the container
- Exposed grid lines as decorative elements
- Portfolio in a tight 4-column grid with thick borders between cards

**Signature Elements:**
- Bold horizontal rules with oversized section labels
- Type that intentionally overflows its container
- Thick-border cards with no rounded corners

**Typography System:**
- Display: Space Grotesk (800) — bold, geometric, modern
- Body: IBM Plex Sans — structured, technical

## Idea C: Warm Monochrome Editorial

**Design Movement:** Contemporary Editorial / Warm Minimalism

**Core Principles:**
- Warm off-white base with dark near-black text — inverted from typical dark portfolios
- Typography-led hierarchy with generous leading and tracking
- Restrained color — only teal accent appears on interactive elements
- Generous whitespace as a signal of confidence

**Color Philosophy:**
- Base: #F5F2EC (warm parchment) — sophisticated, warm, editorial
- Text: #0D0D0D — strong contrast
- Accent: #1A6B5F (deep teal) — professional, distinctive
- Surface: #EDEAE4 — subtle card backgrounds

**Layout Paradigm:**
- Left-rail navigation (sticky) for desktop, hamburger for mobile
- Hero: full-width with large centered display type
- Portfolio: 3-column grid with minimal hover states

**Typography System:**
- Display: Playfair Display (700 italic) — classic editorial
- Body: Geist Sans — modern, clean
